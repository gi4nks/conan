## Prompt (copy & paste)

You are a Senior Full-Stack Engineer.

### Goal

Build a **local-only, Notion-like web application** for personal use.

The app must be:

* **SSR**
* built with **Next.js (App Router)**
* styled **only with DaisyUI** (no custom CSS)
* use **DaisyUI Corporate theme**
* designed for **local usage only**

---

## Tech stack

* Next.js (latest, App Router, SSR)
* React
* Tailwind CSS + **DaisyUI**
* DaisyUI **corporate** theme
* SQLite (local database)
* Node.js runtime

**No external services. No cloud.**

---

## Authentication (very important)

* Simple **basic authentication**
* Single user only
* Username and password stored in `.env`

Example:

```
AUTH_USER=admin
AUTH_PASSWORD=supersecret
```

Rules:

* Login page at `/login`
* Session based authentication (cookie)
* Protect all routes except `/login`
* Logout button clears session
* No OAuth, no email, no registration

---

## Application features (MVP)

### Pages

1. **Login**

   * username
   * password
   * error message on failure

2. **Home / Dashboard**

   * list of pages (documents)
   * button: “New page”

3. **Editor page**

   * simple block-based editor (Notion-like but minimal)
   * supported blocks:

     * heading
     * paragraph
     * bullet list
     * checkbox
     * code block
   * autosave

---

## Data model

Use SQLite.

Tables:

* `pages`

  * id
  * title
  * created_at
  * updated_at

* `blocks`

  * id
  * page_id
  * type
  * content
  * order

---

## UI / UX rules

* Use **only DaisyUI components**
* No custom CSS files
* Clean layout
* Sidebar with pages list
* Top bar with page title + logout
* Corporate theme enabled globally

## Server behavior

* SSR for all pages
